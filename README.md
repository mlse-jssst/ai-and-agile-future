## このレポートについて

これは、日本ソフトウェア科学会科学会 機械学習工学研究会 AIとアジャイルWG の活動の一環として、2030年にAIとアジャイルがどうなっているかを見据えた知見を不定期（3ヶ月めど）でまとめたレポートです。AIを使った開発を試行している・しようとしているアジャイル実践者、および、AI駆動のソフトウェアプロダクト開発を研究しようとしている研究者の双方にとって、今後の指針となることを想定しています。

このレポートは Dave Snowden の Future Backwards を参考にしたバックキャスティングに基づいて作成され、主に以下の3つから構成されます。

1. 2030年頃にAI駆動のアジャイルはどんな姿になっているか（未来像）
2. 現時点で、AIを使ったアジャイルはどのような状態で、どのような課題を抱えているか（現在地）
3. どのようなブレイクスルーがあれば、現在地から未来像に達するか、のロードマップ（未来シナリオ）

注： v0 はこの形式のレポートを作成するまでのたたき台であり、この形式には沿ってはいません。

なお、我々は決して必ず的中する未来を予想しようとは考えていません。AIの進化と移り変わりは激しく、数ヶ月で状況は大きく変わってしまうので、そのような行為自体が不可能であると考えます。このレポートはあくまで現時点での仮説であり、その仮説を提示することで、

- どの方向を向いてAI駆動のアジャイルな開発環境・組織を構築していけば良さそうか
- どういった研究課題があり、何を解決すると今後につながるのか

といった緩やかな指針を、アジャイル実践者、研究者の双方に提供できれば、と考えています。

従って、このレポートは確定的なものではなく、継続的に更新していきます。1つの生きたドキュメントを継続的にアップデートしていき、また、各バージョン間の差分も重要な研究材料として記録していきます。

過去バージョンは Git タグ経由で参照できます（例: [`blob/v0/report/main.md`](https://github.com/mlse-jssst/ai-and-future/blob/v0/report/main.md)）。
