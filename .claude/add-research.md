# 調査資料の追加

現在の執筆バージョンに調査資料（インタビュー記録等）の雛形を追加する。

## 手順

1. 引数から資料の種類とタイトルを取得する（例: `interview 田中氏ヒアリング`）
2. 現在の最新Gitタグを確認し、次のバージョン番号を推定する（タグ v2 が最新なら、作業中は v3 向け）
3. `research/v{次バージョン}/` に資料ファイルを作成する

### インタビューの場合

ファイル名: `interview-{日付}-{短い識別子}.md`

```markdown
# インタビュー: {タイトル}（{今日の日付}）

- 参照レポート: [v{現行}](https://github.com/{org}/{repo}/blob/v{現行}/report/main.md)
- インタビュイー:
- 実施日:
- 実施方法:

## 概要


## 詳細メモ


## レポートへの反映事項

```

### その他の資料の場合

ファイル名: `{種類}-{日付}-{短い識別子}.md` として空のMarkdownファイルを作成する。

4. 作成したファイルのパスを表示して完了を報告する。

## 使い方の例

```
interview 田中氏ヒアリング
```
